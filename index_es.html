Super Coin World (Roblox)

Un juego de plataformas 3D personalizado en Roblox que presenta innovadoras mecánicas de gravedad esférica. Los jugadores pueden explorar biomas planetarios 3D generados procedimentalmente y correr completamente boca abajo. El proyecto muestra una arquitectura cliente-servidor avanzada, simulación de gravedad personalizada y un flujo de trabajo de desarrollo moderno utilizando Rojo, Wally y GitHub Actions CI/CD a través de la API Roblox Open Cloud.

Source Engine Panorama Renderer

Una potente canalización de Python totalmente automatizada para renderizar videos panorámicos 360° VR en 8K de alta fidelidad a partir de juegos con el motor Source como Half-Life 2 y Portal 2. Cuenta con control automático del motor, una robusta plataforma de captura esférica de 22 ángulos y costura perfecta mediante filtros FFmpeg personalizados.

FFmpeg v360 Advanced (Rig Mode)

Una extensión avanzada en C del filtro de video estándar v360 de FFmpeg. Introduce el Modo Rig para unir sin problemas configuraciones multicámara en panorámicas de 360° de alta calidad. Sus características incluyen un algoritmo de Proyección Inversa Ponderada que utiliza interpolación de Hermite y mezcla de prioridad cúbica para una reducción óptima del efecto fantasma.
