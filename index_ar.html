مشاريع شخصية

شبكة TheGamerBay الإعلامية

نظام بيئي متنوع للألعاب يضم أكثر من 1,000,000 مشترك وأكثر من نصف مليار مشاهدة. تُدار وتُشرف عليها عبر أتمتة مخصصة تعتمد على الذكاء الاصطناعي، تتميز الشبكة بقناة رائدة لـ VR 360° بانورامية إلى جانب العديد من القنوات المتخصصة التي تغطي مختلف أنواع الألعاب ومحتوى اللعب عالي الجودة.

زيارة الموقع

Shorts Maker GPU

مكتبة وأداة سطر أوامر (CLI) بلغة بايثون عالية الأداء ومسرّعة بواسطة وحدة معالجة الرسومات (GPU) لإنشاء مقاطع فيديو رأسية (Shorts) تلقائيًا من لقطات اللعب الطويلة. تستخدم الذكاء الاصطناعي لتقييم النشاط الصوتي والمرئي والاكتشاف الذكي للمشاهد عبر NVIDIA VPF و PyTorch.

كاميرا المنظور الثالث الدوارة (Garry's Mod)

إضافة كاميرا منظور فوق الكتف قابلة للتخصيص بشكل كبير للعبة Garry's Mod، مستوحاة من آليات لعبة Haydee. تتميز بآليات تصويب ونظر حقيقية، ووضع هجين للتصويب الذكي، وتتبع ديناميكي ثلاثي الأبعاد لمؤشر التصويب، وانتقالات سينمائية سلسة.

عرض على GitHub

AI Illustrator

أداة قوية لإنشاء رسومات توضيحية سينمائية متسقة وعالية الجودة للقصص تلقائيًا باستخدام نماذج Google Gemini. تتميز بتقسيم دلالي للنص، واتساق ذكي للشخصيات والمواقع من خلال مراجع متعددة الوسائط، وحلقة ضمان جودة متقدمة (LLM-as-a-Judge) للتصحيح الذاتي وإنشاء صور خالية من العيوب.

عرض على GitHub

Haydee AI Outfit Generator GUI

واجهة مستخدم رسومية حديثة ومستقلة لتوليد أزياء مخصصة للعبة Haydee باستخدام الذكاء الاصطناعي. تتميز بتحكم دقيق في الإكساءات (اللون، اللمعان، النتوءات ثلاثية الأبعاد)، ومولد أفكار مدمج للموجهات، وأداة لتجميع التعديلات المتعددة، وحلقة ضمان جودة وتصحيح ذاتي متقدمة للتحقق التلقائي من الخامات المُولدة وإصلاحها.

عرض على GitHub

Super Coin World (Roblox)

لعبة منصات ثلاثية الأبعاد مخصصة في Roblox تتميز بآليات جاذبية كروية مبتكرة تتحدى الجاذبية. يمكن للاعبين استكشاف مناطق أحيائية كوكبية ثلاثية الأبعاد تم إنشاؤها إجرائيًا والركض رأسًا على عقب تمامًا. يعرض المشروع بنية متقدمة للعميل والخادم، ومحاكاة جاذبية مخصصة، وسير عمل تطوير حديث باستخدام Rojo و Wally و GitHub Actions CI/CD عبر واجهة برمجة تطبيقات Roblox Open Cloud.

عرض على GitHub

مُصيّر بانوراما محرك Source (Source Engine Panorama Renderer)

مسار عمل (Pipeline) بايثون قوي ومؤتمت بالكامل لتصيير مقاطع فيديو بانورامية 360 درجة للواقع الافتراضي (VR) بدقة 8K عالية الدقة من ألعاب محرك Source مثل Half-Life 2 و Portal 2. يتميز بتحكم مؤتمت في المحرك، ومنصة التقاط كروية قوية بـ 22 زاوية، ودمج سلس باستخدام فلاتر FFmpeg مخصصة.

عرض على GitHub

FFmpeg v360 Advanced (Rig Mode)

امتداد متقدم مكتوب بلغة C لفلتر فيديو FFmpeg v360 القياسي. يقدم وضع منصة التصوير (Rig Mode) لدمج إعدادات الكاميرات المتعددة بسلاسة في صور بانورامية 360 درجة عالية الجودة. تتضمن الميزات خوارزمية الإسقاط العكسي الموزون باستخدام استيفاء هيرميت ومزج الأولوية التكعيبية لتقليل تأثير الظلال (ghosting) بشكل مثالي.
