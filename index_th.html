อาร์เทม เรียซานอฟ

วิศวกรซอฟต์แวร์และผู้เชี่ยวชาญด้านการบูรณาการ AI เชี่ยวชาญในการสร้างสรรค์ประสบการณ์ดิจิทัลเชิงนวัตกรรม มีความหลงใหลในจุดตัดระหว่างปัญญาประดิษฐ์ (AI) วิศวกรรมซอฟต์แวร์ และวัฒนธรรมเกม

โปรเจกต์ส่วนตัว

เครือข่ายสื่อ TheGamerBay

ระบบนิเวศเกมที่หลากหลายที่มีผู้ติดตามกว่า 1,000,000 คน และยอดวิวมากกว่า ครึ่งพันล้านครั้ง จัดการและดูแลผ่านระบบอัตโนมัติที่ขับเคลื่อนด้วย AI เครือข่ายนี้มีช่องเรือธง 360° VR แบบพาโนรามา พร้อมด้วยช่องเฉพาะทางหลายช่องที่ครอบคลุมประเภทเกมต่างๆ และเนื้อหาเกมเพลย์คุณภาพสูง

เยี่ยมชมเว็บไซต์

Shorts Maker GPU

ไลบรารี Python และเครื่องมือ CLI ประสิทธิภาพสูงที่เร่งความเร็วด้วย GPU สำหรับสร้างคลิปสั้นแนวตั้งจากฟุตเทจเกมเพลย์ขนาดยาวโดยอัตโนมัติ โดยใช้ AI ประเมินโปรไฟล์แอ็กชันของภาพและเสียง พร้อมการตรวจจับฉากอัจฉริยะผ่าน NVIDIA VPF และ PyTorch

ดูบน GitHub

Third Person Rotating Camera (Garry's Mod)

แอดออนกล้องข้ามไหล่ที่สามารถปรับแต่งได้สูงสำหรับ Garry's Mod ซึ่งได้รับแรงบันดาลใจจากกลไกของเกม Haydee มาพร้อมกับ ระบบเล็งเป้าหมายที่แท้จริง, โหมดไฮบริด (Smart Scope), เป้าเล็งติดตามแบบไดนามิก 3 มิติ และการเปลี่ยนมุมกล้องแบบภาพยนตร์ที่ราบรื่น

ดูบน GitHub

AI Illustrator

เครื่องมืออันทรงพลังสำหรับสร้างภาพประกอบเรื่องราวในรูปแบบภาพยนตร์ที่มีความสอดคล้องและคุณภาพสูงโดยอัตโนมัติ โดยใช้โมเดล Gemini ของ Google มาพร้อมการแบ่งข้อความตามความหมาย การรักษาความสม่ำเสมอของตัวละครและสถานที่ผ่านการอ้างอิงแบบ multimodal และ ลูปตรวจสอบคุณภาพ LLM-as-a-Judge ขั้นสูงเพื่อการแก้ไขตัวเองและสร้างภาพที่ไร้ข้อบกพร่อง

ดูบน GitHub

Haydee AI Outfit Generator GUI

อินเทอร์เฟซผู้ใช้แบบกราฟิกที่ทันสมัยสำหรับสร้างชุดแต่งกายแบบกำหนดเองสำหรับเกม Haydee โดยใช้ AI มาพร้อมกับ การควบคุมพื้นผิวอย่างละเอียด (Diffuse, Specular, Normal), เครื่องมือสร้างไอเดียพร้อมต์ในตัว, เครื่องมือจัดกลุ่มม็อด และ ลูปตรวจสอบคุณภาพและแก้ไขตัวเอง ขั้นสูงเพื่อตรวจสอบและแก้ไขพื้นผิวที่สร้างขึ้นโดยอัตโนมัติ

ดูบน GitHub

Super Coin World (Roblox)

เกมแพลตฟอร์ม 3 มิติแบบกำหนดเองใน Roblox ที่มีกลไกแรงโน้มถ่วงทรงกลมที่สร้างสรรค์ ผู้เล่นสามารถสำรวจไบโอมดาวเคราะห์ 3 มิติที่สร้างขึ้นตามขั้นตอนและวิ่งกลับหัวได้ โปรเจกต์นี้นำเสนอสถาปัตยกรรมไคลเอนต์-เซิร์ฟเวอร์ขั้นสูง การจำลองแรงโน้มถ่วงแบบกำหนดเอง และขั้นตอนการทำงานการพัฒนาที่ทันสมัยโดยใช้ Rojo, Wally และ GitHub Actions CI/CD ผ่าน Roblox Open Cloud API

ดูบน GitHub

Source Engine Panorama Renderer

ไปป์ไลน์ Python อัตโนมัติเต็มรูปแบบอันทรงพลังสำหรับการเรนเดอร์ วิดีโอ VR พาโนรามา 360 องศา ความละเอียด 8K คุณภาพสูงจากเกมที่ใช้ Source Engine เช่น Half-Life 2 และ Portal 2 มาพร้อมระบบควบคุมเอนจินอัตโนมัติ แท่นจับภาพทรงกลม 22 มุมที่เสถียร และการต่อภาพที่ไร้รอยต่อโดยใช้ฟิลเตอร์ FFmpeg แบบกำหนดเอง

ดูบน GitHub

FFmpeg v360 Advanced (Rig Mode)

ส่วนขยายที่เขียนด้วยภาษา C ขั้นสูงของฟิลเตอร์วิดีโอ FFmpeg v360 มาตรฐาน มาพร้อม Rig Mode สำหรับการต่อภาพจากกล้องหลายตัวให้เป็นพาโนรามา 360 องศาคุณภาพสูงได้อย่างไร้รอยต่อ มีจุดเด่นคืออัลกอริทึม Weighted Inverse Projection ที่ใช้การอินเทอร์โพเลตแบบ Hermite และการเบลนด์ตามลำดับความสำคัญแบบคิวบิกเพื่อลดภาพซ้อน (ghosting) อย่างเหมาะสมที่สุด

ดูบน GitHub

ช่อง YouTube TheGamerBay

โปรเจกต์สื่อเกมขนาดใหญ่ที่มีผู้ติดตามกว่า 400,000 คน และยอดวิวมากกว่า 200 ล้านครั้ง เชี่ยวชาญด้าน วิดีโอเกมเพลย์ 360° VR ที่สมจริงและประสบการณ์การเล่นเกมแบบพาโนรามาที่ไม่เหมือนใคร ซึ่งผสมผสานเทคโนโลยีและความบันเทิงเข้าด้วยกัน

ดูบน YouTube

ESRB Rating Slate Generator

เครื่องมือสำหรับสร้างป้ายเรตติ้ง ESRB ที่พร้อมออกอากาศ สร้างภาพความละเอียดสูงโดยการดึงข้อมูลอย่างเป็นทางการหรือกรอกรายละเอียดด้วยตนเอง
