Shorts Maker GPU

Third Person Rotating Camera (Garry's Mod)

AI Illustrator

Мощный инструмент для автоматической генерации согласованных и качественных кинематографичных иллюстраций к историям с использованием моделей Google Gemini. Haydee AI Outfit Generator GUI

Современный графический интерфейс для создания пользовательских нарядов для игры Haydee с помощью ИИ. Super Coin World (Roblox)

Кастомный 3D-платформер в Roblox с инновационной механикой сферической гравитации. Игроки могут исследовать процедурно генерируемые 3D-планеты и бегать полностью вверх ногами. Проект демонстрирует продвинутую клиент-серверную архитектуру, симуляцию кастомной гравитации и современный рабочий процесс разработки с использованием Rojo, Wally и GitHub Actions CI/CD через Roblox Open Cloud API.

Source Engine Panorama Renderer

Мощный, полностью автоматизированный Python-пайплайн для рендеринга высококачественных панорамных 360° VR-видео в 8K из игр на движке Source, таких как Half-Life 2 и Portal 2. Включает автоматическое управление движком, надежную систему захвата с 22 ракурсов (сферическую) и бесшовную склейку с использованием кастомных фильтров FFmpeg.

Смотреть на GitHub

FFmpeg v360 Advanced (Rig Mode)

Продвинутое расширение стандартного видеофильтра FFmpeg v360, написанное на C. Добавляет Rig Mode (режим рига) для бесшовной склейки видео с нескольких камер в высококачественные 360° панорамы. Включает алгоритм взвешенной обратной проекции с использованием интерполяции Эрмита и кубического приоритета блендинга для максимального снижения эффекта "призрака" (ghosting).
